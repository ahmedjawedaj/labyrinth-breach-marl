# Deprecated Unity Root Mirrors

This directory contains the legacy root-level Unity mirror trees that were previously present alongside the canonical `unity/Assets/...` source tree.

Archived paths moved here:

- `Scenes/`
- `Prefabs/`
- `Scripts/`
- `Materials/`

These copies are noncanonical and should not be edited for active development.

Canonical source remains under:

- `unity/Assets/`
